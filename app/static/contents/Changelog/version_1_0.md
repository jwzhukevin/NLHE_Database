# v1.0.0 发布说明

本次为项目 1.0.0 正式发布版本。该版本围绕“材料数据管理、检索与可视化”的核心场景，完成了从数据组织、元素智能分析到前后端交互体验的一体化落地，并提供了稳健的只读访问模式与多语言支持。
